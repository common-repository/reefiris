Show your reefiris.com data in the sidebar of your blog.

== Description ==

Adds a widget which queries reefiris.com for all of your tanks and their parameters, and displays them in list form with links back to your tank and data pages on reefiris.com.

== Installation ==

1. Install plugin like any other
1. Head to the widgets config screen for your theme.
1. Make sure to set your reefiris username in the widget options so your data can be queried.
